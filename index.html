Extension of Shadow Dungeon Game

Shadow Dungeon is a fast-paced 2D dungeon crawler built with the Bagel library in Java. Players explore connected rooms, fight projectile-shooting enemies, collect coins, unlock doors, and progress through increasingly challenging areas. Choose between the Robot (extra coins per kill) or the Marine (river-damage immunity), battle enemies like Bullet Kin, and upgrade weapons or health through the in-game store.Controls: WASD to move, mouse to aim/shoot, Space for the store.

Graph Algorithms in C

This project implements key graph algorithms in C, including:

  • Cycle detection using depth-first search (DFS)
  • Dijkstra's shortest path algorithm with a priority queue
  • Graph traversal and path reconstruction

The code demonstrates modular design using structs and functions, allowing for efficient, maintainable, and reusable implementations of graph algorithms. Input data is structured to represent lakes and rivers as nodes and edges.
