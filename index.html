Graph Algorithms in C

This project implements key graph algorithms in C, including:

  • Cycle detection using depth-first search (DFS)
  • Dijkstra's shortest path algorithm with a priority queue
  • Graph traversal and path reconstruction

The code demonstrates modular design using structs and functions, allowing for efficient, maintainable, and reusable implementations of graph algorithms. Input data is structured to represent lakes and rivers as nodes and edges.
